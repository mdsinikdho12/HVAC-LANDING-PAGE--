# HVAC Landing Page

A fully responsive and modern HVAC (Heating, Ventilation, and Air Conditioning) landing page built using **Tailwind CSS**. This project demonstrates a clean UI design, fast performance, and smooth layout adaptability across all screen sizes.

## 🔥 Features

- ✅ Fully responsive design
- 🎨 Modern and clean UI
- ⚡ Built with Tailwind CSS
- 📱 Mobile-first approach
- 🌐 Optimized for all devices
